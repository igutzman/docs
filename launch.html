Colorado Utility Finders Invoice Tool

Welcome! This is an internal tool for generating and submitting invoices to QuickBooks Online.

How to use:

  1. Open the Excel file containing your pothole and ancillary data.
  2. Run the VBA macro to generate the invoice data sheet.
  3. Run the invoice creation macro to send to QuickBooks Online.

Note: This is an internal-use tool only. No public sign-up or login is required.
